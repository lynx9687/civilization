# Online 4x game

## Testing web client locally
To test web client locally build it by running:
```console
./scripts/build_web_client.sh
```
Then run a local http server from `./web-client`:
```console
cd web-client
python3 -m http.server
```
Web client should be reachable at <http://127.0.0.1:8000/> from any browser. You also need to start a local game server.
